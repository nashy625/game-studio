# LinkedIn Launch Post

I started building a small native game studio for my portfolio: Nashy Game Studio Arcade.

The first build is a macOS SpriteKit arcade app with four playable prototypes:

- One Button Samurai: a one-button dash-and-strike action game.
- Micro Dungeon: a compact tactical dungeon crawler.
- Stock Market Survivor: a chaotic trading survival game.
- Neon Pong Royale: a fast first-to-seven paddle duel against an AI opponent.

The goal is to keep shipping small, complete games as real desktop apps, then polish the strongest ones toward Steam-ready releases.

Built with Swift, SpriteKit, and Swift Package Manager.

GitHub: https://github.com/nashy625/game-studio
