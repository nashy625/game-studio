# LinkedIn Launch Post

I started building a small native game studio for my portfolio: Nashy Game Studio Arcade.

The first build is a macOS SpriteKit arcade app with five playable prototypes:

- One Button Samurai: a one-button dash, parry, focus-strike action game.
- Micro Dungeon: a compact tactical dungeon crawler with procedural floors, traps, relics, armor, and upgrades.
- Stock Market Survivor: a chaotic trading survival game with timed buy/sell/hold decisions.
- Neon Pong Royale: a fast first-to-seven paddle duel against an AI opponent.
- Campus Dash: a route-dodging arcade game about collecting notes and making it to class on time.

The goal is to keep shipping small, complete games as real desktop apps, then polish the strongest ones toward Steam-ready releases.

Built with Swift, SpriteKit, and Swift Package Manager.
